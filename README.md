## Quick Start

```python
import pandas as pd
from scipy.stats import pearsonr

# Load quadruplet data
df = pd.read_csv('data/quadruplets_Q47_complete.csv', comment='#')

# Compute correlation
import numpy as np
log_n = np.log(df['n'])
gamma = df['gamma_k']
r, p = pearsonr(log_n, gamma)
print(f"Correlation: r = {r:.3f}, p = {p:.2e}")
```
